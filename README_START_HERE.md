## 🆘 Troubleshooting

### Problem: Build fails
```bash
./gradlew clean
./gradlew build
```

### Problem: Gradle sync fails
```
File → Invalidate Caches → Restart
```

### Problem: API errors
- Check BASE_URL
- Check backend is running
- Check network permissions

### Problem: Hilt errors
- Add @HiltAndroidApp to app class
- Add @AndroidEntryPoint to activities
- Rebuild project
