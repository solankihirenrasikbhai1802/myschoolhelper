## 📊 System Overview

### Users Can
```
👨‍🏫 Teachers:
  • Mark daily attendance
  • Prevent duplicate marking
  • Edit attendance records
  • View historical data
  • Export reports

👨‍🎓 Students:
  • View personal attendance
  • Filter by month
  • See statistics
  • Download reports

🔑 Admins:
  • Manage all users
  • Search attendance
  • Filter by class/date
  • Export comprehensive reports
  • Generate statistics
```

### Technical Architecture
```
Frontend:  React + Context API + Axios
Routing:   React Router v6
Styling:   Tailwind CSS (responsive)
Backend:   Express.js (Node.js)
Database:  MongoDB + Mongoose
Auth:      JWT tokens (7-day expiry)
Security:  bcryptjs + RBAC + validation
```

## 📊 API Endpoints (23 Total)

```
Authentication (3):
  POST /api/auth/login
  POST /api/auth/register
  GET /api/auth/verify

Classes (4):
  GET /api/classes
  GET /api/classes/:classId/sections
  POST /api/classes
  POST /api/classes/sections

Students (5):
  GET /api/students
  POST /api/students
  GET /api/students/:studentId
  PUT /api/students/:studentId
  DELETE /api/students/:studentId

Attendance (7):
  POST /api/attendance/mark
  PUT /api/attendance/update
  GET /api/attendance/check-existing
  GET /api/attendance/view
  GET /api/student/attendance/records
  GET /api/student/attendance/stats
  GET /api/admin/attendance/reports

Admin (4):
  GET /api/admin/users
  POST /api/admin/users
  PUT /api/admin/users/:userId
  DELETE /api/admin/users/:userId
```
